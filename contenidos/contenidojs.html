Contenido de JS

Javascript es un lenguaje de programación interpretado, basado en el estándar ECMAScript. Su naturaleza más conocida es la parte del cliente (client-side), integrada en los navegadores web, donde se puede utilizar para mejorar la interfaz de usuario, hacer las páginas web más dinámicas o construir aplicaciones o sitios web más complejos y potentes.

Lenguaje JS es una página de documentación y herramientas dedicadas al aprendizaje y perfeccionamiento de la programación web con Javascript.
